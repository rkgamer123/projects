## Scientific Calculator (Python)
A GUI-based scientific calculator built in Python using Tkinter.
The calculator evaluates mathematical expressions, supports advanced functions, and shows intermediate computation steps.

## Features
Arithmetic operations (+, −, ×, ÷, %, powers)
Trigonometric functions (sin, cos, tan, etc.)
Logarithms and roots
Factorial support
Constants: π and e
Degree and radian modes
Step-by-step evaluation display
Error handling for invalid expressions

## Technologies Used

Python
Tkinter (GUI)
Math library
Expression parsing logic

## How to Run
Make sure Python is installed

Run:

    python3 calculator.py
